# trdoc

translater

[Language List](https://cloud.google.com/translate/docs/languages)


## Usage
```bash
Usage of trdoc:
  -s string
    	source language. ex. en, ja ...
  -t string
    	target language. ex. ja, en ...
  -version
    	Print version information and quit.
```
